## neovim

------

### 1. 模式

| 模式   | 作用               | 进入方式       |
| ------ | ------------------ | -------------- |
| Normal | 操作文本、移动光标 | Esc            |
| Insert | 输入文本           | i / a / o      |
| Visual | 选择文本           | v / V / Ctrl+v |

------

### 2. 光标移动

#### 基础移动

| 按键 | 作用 |
| ---- | ---- |
| h    | 左   |
| j    | 下   |
| k    | 上   |
| l    | 右   |

------

#### 单词移动

| 按键 | 作用             |
| ---- | ---------------- |
| w    | 到下一个单词开头 |
| b    | 到上一个单词开头 |
| e    | 到当前单词结尾   |

------

#### 行内与文件移动

| 按键   | 作用         |
| ------ | ------------ |
| 0      | 行首         |
| $      | 行尾         |
| gg     | 文件开头     |
| G      | 文件结尾     |
| {n}G   | 跳到第 n 行  |
| Ctrl+g | 显示当前位置 |

------

#### 屏幕定位

| 按键 | 作用     |
| ---- | -------- |
| H    | 屏幕顶部 |
| M    | 屏幕中间 |
| L    | 屏幕底部 |

------

### 3. 操作符（Operator）

| 操作符 | 作用                       |
| ------ | -------------------------- |
| d      | 删除                       |
| c      | 修改（删除并进入插入模式） |
| y      | 复制                       |

规则：operator + motion

------

### 4. 操作组合

#### 删除

| 命令 | 作用                                   |
| ---- | -------------------------------------- |
| x    | 删除当前字符                           |
| dw   | 删除到下一个单词开头（会把空格一起删） |
| de   | 删除到当前单词结尾（不删后面的空格）   |
| d$   | 删除到行尾                             |
| dd   | 删除整行                               |

------

#### 修改

| 命令 | 作用           |
| ---- | -------------- |
| cw   | 修改一个单词   |
| ce   | 修改到单词结尾 |
| ciw  | 修改整个单词   |
| cc   | 修改整行       |

------

#### 复制与粘贴

| 命令 | 作用         |
| ---- | ------------ |
| yw   | 复制一个单词 |
| yy   | 复制整行     |
| p    | 在后粘贴     |
| P    | 在前粘贴     |

------

### 5. 数字（Count）

| 写法 | 作用         |
| ---- | ------------ |
| 2dd  | 删除两行     |
| d2w  | 删除两个单词 |
| 2w   | 移动两个单词 |

规则：数字作用于右侧命令或移动

------

### 6. 插入模式

| 按键 | 作用             |
| ---- | ---------------- |
| i    | 光标前插入       |
| a    | 光标后插入       |
| o    | 下一行插入       |
| O    | 上一行插入       |
| Esc  | 返回 Normal 模式 |

------

### 7. 撤销与重复

| 按键   | 作用           |
| ------ | -------------- |
| u      | 撤销           |
| Ctrl+r | 重做           |
| .      | 重复上一次操作 |

------

### 8. 搜索

| 命令  | 作用                 |
| ----- | -------------------- |
| /text | 向下搜索             |
| ?text | 向上搜索             |
| n     | 下一个匹配           |
| N     | 上一个匹配           |
| *     | 搜索当前单词（向下） |
| #     | 搜索当前单词（向上） |

------

### 9. 替换

#### 单字符替换

| 按键 | 作用         |
| ---- | ------------ |
| r    | 替换当前字符 |

------

#### 文本替换

| 命令          | 作用           |
| ------------- | -------------- |
| :s/old/new    | 当前行替换     |
| :s/old/new/g  | 当前行全部替换 |
| :%s/old/new/g | 全文件替换     |

------

### 10. 括号匹配

| 按键 | 作用         |
| ---- | ------------ |
| %    | 跳转匹配括号 |

------

### 11. Visual 模式

| 按键   | 作用     |
| ------ | -------- |
| v      | 字符选择 |
| V      | 行选择   |
| Ctrl+v | 块选择   |

------

### 12. 文件操作

| 命令        | 作用       |
| ----------- | ---------- |
| :w          | 保存       |
| :q          | 退出       |
| :wq         | 保存并退出 |
| :q!         | 强制退出   |
| :e filename | 打开文件   |
| :w filename | 另存为     |

------

### 13. 帮助系统

| 命令            | 作用           |
| --------------- | -------------- |
| :help           | 打开帮助       |
| :help {command} | 查看某命令帮助 |

------

### 14. 核心思想

| 概念 | 说明                     |
| ---- | ------------------------ |
| 模式 | Normal / Insert / Visual |
| 移动 | 控制光标位置             |
| 操作 | d / c / y                |
| 组合 | operator + motion        |
| 重复 | 使用数字和 . 提高效率    |
