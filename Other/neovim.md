## neovim

------

### 1. 模式

| 模式   | 作用               | 进入方式       |
| ------ | ------------------ | -------------- |
| Normal | 操作文本、移动光标 | Esc            |
| Insert | 输入文本           | i / a / o      |
| Visual | 选择文本           | v / V / Ctrl+v |

------

### 2. 光标移动

#### 基础移动

| 按键 | 作用 |
| ---- | ---- |
| h    | 左   |
| j    | 下   |
| k    | 上   |
| l    | 右   |

------

#### 单词移动

| 按键 | 作用             |
| ---- | ---------------- |
| w    | 到下一个单词开头 |
| b    | 到上一个单词开头 |
| e    | 到当前单词结尾   |

------

#### 行内与文件移动

| 按键   | 作用         |
| ------ | ------------ |
| 0      | 行首         |
| $      | 行尾         |
| gg     | 文件开头     |
| G      | 文件结尾     |
| {n}G   | 跳到第 n 行  |
| Ctrl+g | 显示当前位置 |

------

#### 屏幕定位

| 按键 | 作用     |
| ---- | -------- |
| H    | 屏幕顶部 |
| M    | 屏幕中间 |
| L    | 屏幕底部 |

------

### 3. 操作符

| 操作符 | 作用                       |
| ------ | -------------------------- |
| d      | 删除                       |
| c      | 修改（删除并进入插入模式） |
| y      | 复制                       |

规则：operator + motion

------

### 4. 操作组合

#### 删除

| 命令 | 作用                                   |
| ---- | -------------------------------------- |
| x    | 删除当前字符                           |
| dw   | 删除到下一个单词开头（会把空格一起删） |
| de   | 删除到当前单词结尾（不删后面的空格）   |
| d$   | 删除到行尾                             |
| dd   | 删除整行                               |

| 类型             | 命令        | 说明             |
| ---------------- | ----------- | ---------------- |
| 段落（不含空行） | `dip`       | inner paragraph  |
| 段落（含空行）   | `dap`       | around paragraph |
| 行范围删除       | `:10,20d`   | 按行号           |
| 当前行到某行     | `:.,20d`    | 向下             |
| 某行到当前行     | `:10,.d`    | 向上             |
| 可视删除         | `V + d`     | 手动选           |
| 关键词删除       | `:g/xxx/d`  | 全局删除         |
| 代码块删除       | `di{ / da{` | 结构删除         |

------

#### 修改

| 命令 | 作用           |
| ---- | -------------- |
| cw   | 修改一个单词   |
| ce   | 修改到单词结尾 |
| ciw  | 修改整个单词   |
| cc   | 修改整行       |

------

#### 复制与粘贴

| 类型 | 命令      | 作用                 | 说明                         |
| ---- | --------- | -------------------- | ---------------------------- |
| 单词 | `yw`      | 从光标复制到单词末尾 | 按字符到词尾                 |
| 单词 | `yiw`     | 复制整个单词         | inner word（推荐）           |
| WORD | `yiW`     | 复制整个 WORD        | 包含 `dp[i][j]` 这种符号整体 |
| WORD | `yaW`     | 复制 WORD + 边界符   | 含空格/符号                  |
| 行   | `yy`      | 复制整行             | 最常用                       |
| 可视 | `v + y`   | 可视模式复制         | 自由选择                     |
| 可视 | `viw + y` | 选中单词复制         | 更精确                       |

段落操作:

| 类型 | 命令  | 作用                     | 说明             |
| ---- | ----- | ------------------------ | ---------------- |
| 段落 | `yip` | 复制当前段落（不含空行） | inner paragraph  |
| 段落 | `yap` | 复制当前段落（含空行）   | around paragraph |

行范围操作:

| 类型         | 命令      | 作用                     | 说明           |
| ------------ | --------- | ------------------------ | -------------- |
| 行范围       | `:10,20y` | 复制第10~20行            | 精确范围       |
| 当前到末尾   | `:.,$y`   | 当前行到文件末尾         | `$` = 最后一行 |
| 全文 ⭐       | `:%y+`    | 复制整个文件到系统剪贴板 | 最标准写法     |
| 全文（备用） | `:1,$y+`  | 复制整个文件             | 等价写法       |

剪贴板相关:

| 类型       | 命令   | 作用                   | 说明             |
| ---------- | ------ | ---------------------- | ---------------- |
| 系统剪贴板 | `"+yy` | 复制当前行到系统剪贴板 | GUI Vim / Neovim |
| 系统剪贴板 | `"+y`  | 复制选中内容           | 通用             |

| 命令 | 作用         |
| ---- | ------------ |
| `p`  | 在光标后粘贴 |
| `P`  | 在光标前粘贴 |

| 概念   | 含义                     | 示例     |
| ------ | ------------------------ | -------- |
| `word` | 变量级（字母数字下划线） | dp       |
| `WORD` | 表达式级（以空格分隔）   | dp[i][j] |

------

### 5. 数字（Count）

| 写法 | 作用         |
| ---- | ------------ |
| 2dd  | 删除两行     |
| d2w  | 删除两个单词 |
| 2w   | 移动两个单词 |

规则：数字作用于右侧命令或移动

------

#### 文本替换

### 12. 文件操作

| 命令        | 作用       |
| ----------- | ---------- |
| :w          | 保存       |
| :q          | 退出       |
| :wq         | 保存并退出 |
| :q!         | 强制退出   |
| :e filename | 打开文件   |

------

### 14. 核心思想

| 概念 | 说明                     |
| ---- | ------------------------ |
| 模式 | Normal / Insert / Visual |
| 移动 | 控制光标位置             |
| 操作 | d / c / y                |
| 组合 | operator + motion        |
| 重复 | 使用数字和 . 提高效率    |
