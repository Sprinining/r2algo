
## 分治法

分治法（Divide and Conquer）是计算机科学中最著名的算法设计范式之一。它的核心思想正如其名：“分而治之”。通过将一个复杂的大问题分解为若干个规模较小、结构相同的子问题，逐个解决这些子问题，最后将子问题的解合并，从而得到原问题的解。

### 一、 分治法的核心步骤

分治算法在实现上通常采用**递归 (Recursion)**，其求解过程主要包含以下三个核心步骤：

1. **分解 (Divide)**：将原问题分解为若干个规模较小、相互独立且与原问题形式相同的子问题。
2. **解决 (Conquer)**：若子问题规模足够小（达到边界条件/基准情形），则直接求解；否则，继续递归地解决各个子问题。
3. **合并 (Combine)**：将各个子问题的解合并，组合出原问题的解。

### 二、 分治法的适用条件

一个问题能否使用分治法解决，通常取决于它是否同时满足以下四个特征：

1. **可分解性**：该问题能够分解为若干个规模较小的相同子问题。
2. **子问题同质性**：分解出的子问题与原问题形式相同，这意味着可以使用相同的递归逻辑。
3. **子问题独立性**：各个子问题之间是独立的，即**子问题之间不包含公共的子子问题**。（*注意：如果子问题重叠，通常更适合使用动态规划*）。
4. **可合并性**：利用子问题的解可以合并出原问题的解，且合并操作的复杂度不能过高。

### 三、 经典应用场景与算法

分治法在诸多经典算法中有着广泛的应用：

#### 1. 二分查找 (Binary Search)

- **思想**：在一份单调有序的数组中，每次取中间元素与目标值比较，将查找范围缩小一半。
- **分治体现**：将规模为 $N$ 的查找问题，分解为规模为 $N/2$ 的子问题。由于每次只需关心其中一个子问题，不需要合并。
- **时间复杂度**：$O(\log n)$

#### 2. 归并排序 (Merge Sort)

- **思想**：将一个无序数组分成左右两个子数组，分别进行排序，然后将两个有序的子数组合并成一个完全有序的数组。
- **分治体现**：
  - **分**：找到数组中点，分为左右两半。
  - **治**：递归对左右两半进行归并排序。
  - **合**：双指针线性扫描，将两个有序数组合并（Merge 操作）。
- **时间复杂度**：$O(n \log n)$

#### 3. 快速排序 (Quick Sort)

- **思想**：选择一个基准元素（Pivot），通过一趟排序将待排记录分割成独立的两部分，其中一部分记录的关键字均比另一部分小，再分别对这两部分进行排序。
- **分治体现**：
  - **分**：以 Pivot 为界，把数组划分为“小于 Pivot”和“大于 Pivot”两部分。
  - **治**：对这两部分独立进行递归快排。
  - **合**：因为是原地划分，子问题排好序后，整体自然有序，无需显式合并。
- **时间复杂度**：平均 $O(n \log n)$，最坏 $O(n^2)$

#### 4. 其他经典问题

- **大整数乘法 (Karatsuba 算法)**：将两个 $n$ 位大整数乘法转化为 3 次 $n/2$ 位整数乘法，将复杂度从 $O(n^2)$ 降到 $O(n^{\log_2 3}) \approx O(n^{1.585})$。
- **Strassen 矩阵乘法**：将两个 $n \times n$ 矩阵乘法转化为 7 次 $n/2 \times n/2$ 矩阵乘法，将复杂度从 $O(n^3)$ 降到 $O(n^{\log_2 7}) \approx O(n^{2.807})$。
- **汉诺塔问题 (Hanoi)**：将 $n$ 个盘子从 A 柱移到 C 柱，转化为先将 $n-1$ 个盘子移到 B 柱，再移动最大的盘子，最后将 $n-1$ 个盘子从 B 柱移到 C 柱。

### 四、 分治法的复杂度分析（主定理）

分析分治算法的时间复杂度时，我们通常会写出其递归方程：

$$T(n) = aT\left(\frac{n}{b}\right) + f(n)$$

- $a$：生成的子问题个数 ($a \ge 1$)
- $n/b$：每个子问题的规模 ($b > 1$)
- $f(n)$：分解与合并子问题所消耗的时间

根据**主定理 (Master Theorem)**，我们可以快速推导其时间复杂度：

1. 若 $f(n) < O(n^{\log_b a})$，则 $T(n) = O(n^{\log_b a})$ （子问题求解占主导）
2. 若 $f(n) = O(n^{\log_b a})$，则 $T(n) = O(n^{\log_b a} \log n)$ （各层消耗均匀）
3. 若 $f(n) > O(n^{\log_b a})$，则 $T(n) = O(f(n))$ （分解与合并占主导）

*例如归并排序：$T(n) = 2T(n/2) + O(n)$，此时 $a=2, b=2, f(n)=O(n)$。由于 $n^{\log_2 2} = n^1$，符合第二种情况，故 $T(n) = O(n \log n)$。*

### 五、 分治法 vs 动态规划 vs 贪心算法

| **算法策略** | **子问题关系**                     | **求解顺序**                   | **核心关注点**                     |
| ------------ | ---------------------------------- | ------------------------------ | ---------------------------------- |
| **分治法**   | 子问题相互**独立**，互不重叠       | 自顶向下 (Top-down)            | 侧重于问题的结构分解与合并         |
| **动态规划** | 子问题**重叠**，存在大量重复子问题 | 自底向上 (Bottom-up) 或 备忘录 | 侧重于重叠子问题的复用与最优子结构 |
| **贪心算法** | 子问题相互独立                     | 自顶向下 (Top-down)            | 每一阶段都做出当前局部的最优选择   |

### 六、 总结

分治法是一种**自顶向下**的解决问题策略。它不仅能帮助我们显著降低算法的时间复杂度（例如将 $O(n^2)$ 优化至 $O(n \log n)$），更是一种极其重要的现代计算思维。在面对庞大、复杂的系统工程时，将其拆解为模块化、高内聚、低耦合的子模块，同样也是分治思想的具象体现。


### [912. 排序数组](https://leetcode.cn/problems/sort-an-array/)

```c++
#include <iostream>
#include <vector>

using namespace std;

class Solution {
public:
    // 分治-治
    void merge(vector<int> &arr, int left, int mid, int right, vector<int> &temp) {
        // [left, mid]和[mid+1, right]两个有序数组
        int i = left;
        int j = mid + 1;
        int index = 0;
        while (i <= mid && j <= right) {
            if (arr[i] < arr[j])
                temp[index++] = arr[i++];
            else
                temp[index++] = arr[j++];
        }
        // 剩余元素直接放入temp
        while (i <= mid) temp[index++] = arr[i++];
        while (j <= right) temp[index++] = arr[j++];
        // 放回原数组
        index = 0;
        while (left <= right) arr[left++] = temp[index++];
    }

    // 分治-分
    // T(n) = 2 * T(n/2) + O(n)
    // a = 2, b = 2, c = 1, 根据 master 公式，时间复杂度为 O(n * logn)
    // 空间复杂度 O(n)
    void divide(vector<int> &arr, int left, int right, vector<int> &temp) {
        if (left >= right) return;
        int mid = left + ((right - left) >> 2);
        // 左边归并排序
        divide(arr, left, mid, temp);
        // 右边归并排序
        divide(arr, mid + 1, right, temp);
        // 合并两个有序序列
        merge(arr, left, mid, right, temp);
    }

    // 递归版归并排序
    void mergeSort(vector<int> &arr) {
        vector<int> temp(arr.size());
        divide(arr, 0, arr.size() - 1, temp);
    }

    vector<int> sortArray(vector<int> &nums) {
        mergeSort(nums);
        return nums;
    }
};
```

#include <iostream>

class Solution {
public:
    // 分治-治
    void merge(vector<int> &arr, int left, int mid, int right, vector<int> &temp) {
        // [left, mid]和[mid+1, right]两个有序数组
        int i = left;
        int j = mid + 1;
        int index = 0;
        while (i <= mid && j <= right) {
            if (arr[i] < arr[j])
                temp[index++] = arr[i++];
            else
                temp[index++] = arr[j++];
        }
        // 剩余元素直接放入temp
        while (i <= mid) temp[index++] = arr[i++];
        while (j <= right) temp[index++] = arr[j++];
        // 放回原数组
        index = 0;
        while (left <= right) arr[left++] = temp[index++];
    }


    // 非递归版归并排序
    void mergeSort(vector<int> &arr) {
        vector<int> temp(arr.size());
        int n = arr.size();
        // 执行 O(logn) 次
        for (int left, mid, right, step = 1; step < n; step <<= 1) {
            left = 0;
            // O(n)
            while (left < n) {
                mid = left + step - 1;
                // 没有右侧
                if (mid + 1 >= n) break;
                // 求右边界
                right = min(left + (step << 1) - 1, n - 1);
                merge(arr, left, mid, right, temp);
                // 找下一组
                left = right + 1;
            }
        }
    }

    vector<int> sortArray(vector<int> &nums) {
        mergeSort(nums);
        return nums;
    }
};

- 思考一个问题在大范围上的答案，是否等于：左部分答案 + 右部分答案 + 跨越左右产生的答案
- 计算跨越左右产生的答案时，如果加上左右两侧都有序，能不能简化计算

### [计算数组的小和](https://www.nowcoder.com/practice/edfe05a1d45c4ea89101d936cac32469)

```c++
#include <iostream>
#include <vector>

using namespace std;

vector<int> temp;

void merge(vector<int> &arr, int left, int mid, int right) {
    int i = left;
    int j = mid + 1;
    int index = 0;
    while (i <= mid && j <= right) {
        if (arr[i] <= arr[j]) {
            temp[index++] = arr[i++];
        } else {
            temp[index++] = arr[j++];
        }
    }
    while (i <= mid) temp[index++] = arr[i++];
    while (j <= right) temp[index++] = arr[j++];
    index = 0;
    while (left <= right) arr[left++] = temp[index++];
}

// 返回小和，且把数组这一段变成有序
long divide(vector<int> &arr, int left, int right) {
    if (left >= right) return 0;
    int mid = left + ((right - left) >> 1);
    // 分别计算两侧的小和
    long leftSum = divide(arr, left, mid);
    long rightSum = divide(arr, mid + 1, right);
    // 两部分都已经有序
    long midSum = 0;
    for (int l = left, r = mid + 1, sum = 0; r <= right; r++) {
        // 找到第一个大于 arr[r] 的位置
        while (l <= mid && arr[l] <= arr[r]) {
            sum += arr[l++];
        }
        midSum += sum;
    }

    merge(arr, left, mid, right);
    return leftSum + midSum + rightSum;
}

int main() {
    int n;
    cin >> n;
    vector<int> arr(n);
    temp.resize(n);
    for (int i = 0; i < n; ++i)
        cin >> arr[i];
    cout << divide(arr, 0, n - 1);
}
```

### [493. 翻转对](https://leetcode.cn/problems/reverse-pairs/)

```c++
#include <iostream>
#include <vector>

using namespace std;

class Solution {
public:
    vector<int> temp;

    void merge(vector<int> &arr, int left, int mid, int right) {
        int i = left;
        int j = mid + 1;
        int index = 0;
        while (i <= mid && j <= right) {
            if (arr[i] <= arr[j]) {
                temp[index++] = arr[i++];
            } else {
                temp[index++] = arr[j++];
            }
        }
        while (i <= mid) temp[index++] = arr[i++];
        while (j <= right) temp[index++] = arr[j++];
        index = 0;
        while (left <= right) arr[left++] = temp[index++];
    }

    int divide(vector<int> &nums, int left, int right) {
        if (left >= right) return 0;
        int mid = left + ((right - left) >> 1);
        int leftSum = divide(nums, left, mid);
        int rightSum = divide(nums, mid + 1, right);

        int midSum = 0;
        for (int l = left, r = mid + 1, sum = 0; l <= mid; ++l) {
            while (r <= right && ((long) nums[l] > (long) 2 * nums[r])) {
                sum++;
                r++;
            }
            midSum += sum;
        }

        merge(nums, left, mid, right);
        return leftSum + midSum + rightSum;
    }

    int reversePairs(vector<int> &nums) {
        temp.resize(nums.size());
        return divide(nums, 0, nums.size() - 1);
    }
};
```
